🛒 E-Commerce Application (Full-Stack)
📌 Overview

This project is a full-stack e-commerce application developed during my internship. It follows an MVC architecture and simulates a real-world online shopping platform with user and admin functionalities.

🚀 Features
User authentication (login/register)
Product listing and management
Admin panel for product control
Shopping cart functionality
Basic payment flow simulation
RESTful API structure
🛠️ Technologies Used
Backend: Node.js, Express.js
Database: MongoDB, Mongoose
Architecture: MVC pattern
🧠 What I Did
Designed and implemented backend architecture using MVC
Built REST APIs for products, users, and cart operations
Implemented authentication and authorization
Managed database schemas with Mongoose
Improved performance and organized project documentation (PRD/SRS)
📚 What I Learned
Structuring scalable backend applications
Working with NoSQL databases (MongoDB)
Building real-world REST APIs
Understanding full-stack development workflows
